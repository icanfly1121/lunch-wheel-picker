# 今天吃什麼｜午餐轉盤

這是一個午餐選擇器。使用者可以瀏覽全部午餐品項，也可以用轉盤隨機選出今天要吃什麼。

## 功能

- 午餐轉盤隨機選擇品項
- 全部品項瀏覽
- 搜尋品項 / 店家 / 備註
- 新增與刪除自訂午餐品項
- localStorage 保存資料
- React Router 多頁面

## 使用技術

- Vite + React
- React Router
- useState / useEffect
- props / component 拆分
- map / filter
- localStorage
- CSS animation / RWD

## 執行方式

```bash
npm install
npm run dev
```

開啟終端機顯示的網址，通常是：

```txt
http://localhost:5173/
```
